  13. def hlpOrganizeAttributes(atts: Attributes, ofInterest: ArrayBuffer[String]): Any

    hlpOrganizeAttributes is called from each of the mkPmml* 'make' fcns in order to get the attributes from the PMML in the order of the constructor.

    hlpOrganizeAttributes is called from each of the mkPmml* 'make' fcns in order to get the attributes from the PMML in the order of the constructor. If the attribute, especially common on the optional attributes, is not present, a value of 'None' is returned in its place.

    At some point, we can dress this up by utilizing the defaults from the xsd instead (meaning that another ArrayBuffer would be supplied here with the appropriate defaults for each attribute).
